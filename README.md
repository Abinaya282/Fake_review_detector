# Fake_review_detector
A machine learning project that classifies reviews as fake or genuine using TF-IDF and Logistic Regression, deployed with Streamlit.
# Fake Review Detector

This project detects whether a review is fake or genuine using Machine Learning.

## Features
- NLP using TF-IDF
- Logistic Regression model
- Streamlit web app

## How to run
1. python train.py
2. python -m streamlit run app.py
