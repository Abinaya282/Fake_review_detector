# Fake_review_detector
A machine learning project that classifies reviews as fake or genuine using TF-IDF and Logistic Regression, deployed with Streamlit.
# 🕵️ Fake Review Detector

This project detects whether a review is fake or genuine using Machine Learning and NLP.

## 🚀 Features
- Text classification using TF-IDF
- Logistic Regression model
- Streamlit web application
- Real-time prediction

## 🧠 Technologies Used
- Python
- Scikit-learn
- Streamlit
- Pandas

## ▶️ How to Run

1. Train the model:
   python train.py

2. Run the app:
   python -m streamlit run app.py

## 📌 Example

Input: "This product is amazing"  
Output: Genuine Review

Input: "Fake product scam"  
Output: Fake Review
